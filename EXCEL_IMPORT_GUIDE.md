# Excel批量导入模板说明

## 文件格式要求

系统支持通过Excel批量导入产品数据。Excel文件必须按照以下格式组织：

### 必需的列

1. **货号** - 产品唯一编号，必填
2. **产品名称** - 产品名称，必填

### 可选的列

3. **条码** - 产品条形码
4. **型号** - 产品型号
5. **规格** - 产品规格
6. **单位** - 计量单位（如：件、个、箱），默认为"件"
7. **零售价** - 零售价格，默认为0
8. **批发价** - 批发价格，默认为0
9. **批发最小数量** - 批发起订数量，默认为1
10. **库存** - 库存数量，默认为0
11. **描述** - 产品描述

## 创建Excel文件的步骤

### 方法一：使用Excel创建

1. 打开 Microsoft Excel 或 WPS 表格
2. 在第一行输入列名（表头）：
   - A1: 货号
   - B1: 条码
   - C1: 产品名称
   - D1: 型号
   - E1: 规格
   - F1: 单位
   - G1: 零售价
   - H1: 批发价
   - I1: 批发最小数量
   - J1: 库存
   - K1: 描述

3. 从第二行开始填写产品数据
4. 保存为 `.xlsx` 或 `.xls` 格式

### 方法二：使用CSV创建

1. 打开文本编辑器（如记事本）
2. 输入以下内容（使用逗号分隔）：

```csv
货号,条码,产品名称,型号,规格,单位,零售价,批发价,批发最小数量,库存,描述
P001,6901234567890,洗衣液 - 3L,XL-001,3L/瓶,瓶,29.90,25.00,10,100,超浓缩洗衣液，去污力强
P002,6901234567891,洗衣液 - 2L,XL-002,2L/瓶,瓶,19.90,16.50,15,200,超浓缩洗衣液，家庭装
P003,6901234567892,洗洁精 - 1.5L,DJ-001,1.5L/瓶,瓶,9.90,8.50,20,300,强力去油，易漂洗
```

3. 保存为 `.csv` 格式（使用UTF-8编码）

### 方法三：使用模板文件

如果系统提供了示例Excel文件，可以直接：

1. 下载示例Excel文件
2. 打开并编辑示例文件
3. 保存修改后的文件

## 注意事项

### 数据格式

1. **货号和产品名称**
   - 不能为空
   - 货号必须唯一，重复的货号会被跳过

2. **价格字段**
   - 只能输入数字
   - 支持小数（如：29.90）
   - 不能包含货币符号（¥、$等）

3. **数量字段**
   - 只能输入整数
   - 不能为负数

4. **文本字段**
   - 支持中文、英文、数字
   - 不能包含特殊符号（如换行符、制表符）

### 限制

1. **文件大小**
   - 最大 10MB

2. **记录数量**
   - 建议不超过 1000 条
   - 超过可能导致导入超时

3. **图片**
   - Excel中不包含图片
   - 导入后可以在后台添加产品图片

### 常见错误

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| "缺少必填字段（货号或产品名称）" | 某行的货号或产品名称为空 | 检查Excel文件，确保所有行的货号和产品名称都有值 |
| "货号 XXX 已存在" | 货号重复 | 修改货号使其唯一，或使用"编辑"功能更新现有产品 |
| "价格格式错误" | 价格字段包含非数字字符 | 删除货币符号，只保留数字和小数点 |
| "文件格式不支持" | 文件不是Excel或CSV格式 | 确保文件扩展名为 .xlsx、.xls 或 .csv |

## 导入步骤

1. **准备Excel文件**
   - 按照上述要求创建Excel文件
   - 确保数据格式正确

2. **登录后台系统**
   - 使用管理员账号登录

3. **进入批量导入页面**
   - 点击"产品管理" → "批量导入"

4. **上传文件**
   - 选择准备好的Excel文件
   - 点击"开始导入"

5. **查看导入结果**
   - 系统会显示导入统计：
     - 成功导入数量
     - 失败数量
     - 错误详情

6. **处理错误**
   - 如有错误，根据错误信息修正Excel文件
   - 重新上传修正后的文件

7. **添加图片**
   - 导入成功后，在"产品管理"中找到导入的产品
   - 点击"编辑"按钮
   - 上传产品图片或输入网络图片URL

## 导入后操作

1. **验证数据**
   - 在"产品管理"中查看导入的产品
   - 检查价格、库存等信息是否正确

2. **添加分类**
   - 如需要，为产品分配分类

3. **上传图片**
   - 上传产品图片
   - 或从网络URL下载图片

4. **设置库存**
   - 更新产品实际库存数量

## 导入模板下载

在系统的"批量导入"页面中，可以下载示例Excel文件作为模板。

## 技术支持

如遇到导入问题，请：

1. 检查Excel文件格式是否符合要求
2. 查看错误提示信息
3. 参考本文档的注意事项
4. 联系技术支持
