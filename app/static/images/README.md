# 静态资源说明

本目录用于存储系统中的静态图片资源。

## 产品图片

### 网络图片资源

系统初始化时，测试产品使用的是来自Unsplash的网络图片URL。这些图片是公开可用的高质量图片，非常适合用于演示和测试。

使用网络图片的优势：
1. 无需下载和存储大量图片文件
2. 图片质量高，视觉效果好
3. 部署后无需额外配置即可看到完整效果

### 本地图片

如果需要使用本地图片，请将图片文件放在本目录下，然后在产品管理中上传或使用。

#### 占位图片

本目录包含两个占位图片文件，用于产品没有图片时显示：
- `product-placeholder.jpg` - JPG格式占位图
- `product-placeholder.svg` - SVG格式占位图

## 图片上传路径

产品图片上传后会保存到：
- 开发环境：`app/static/uploads/products/`
- 生产环境：系统配置的 `UPLOAD_FOLDER` 目录

## 自定义测试图片

如果要添加自定义测试图片：

1. 将图片文件复制到本目录（`app/static/images/`）
2. 图片格式支持：JPG, JPEG, PNG, GIF, WEBP
3. 建议图片尺寸：400x400 像素
4. 建议图片大小：小于 500KB

## 网络图片源

当前使用的测试图片来自：
- Unsplash (https://unsplash.com)

所有图片均为免费商用图片，可以放心用于演示和测试。
